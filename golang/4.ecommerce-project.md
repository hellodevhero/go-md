## Domain Model

```
User ──────────────── Order ─────────────── OrderItem
 │                      │                       │
 │                      │                    Product
 │                   Address                    │
 │                                          Category
Cart ──────────────── CartItem                  │
                          │                 ProductImage
                       Product
```

The core domain relationships work as follows: a **User** places **Orders**, each
containing one or more **OrderItems** that snapshot the **Product** at purchase time
(name, price, SKU) so the order record stays accurate even if the catalog changes later.
**Products** belong to **Categories**, which form a tree via `parent_id` — allowing
top-level categories like "Clothing" with children like "Shirts" and "Pants".
The **Cart** is an ephemeral structure stored in Redis (not PostgreSQL) because most
carts are abandoned and never convert to orders; Redis TTL handles automatic cleanup.
When a cart does convert, the checkout flow creates an Order, a Stripe payment session,
and clears the cart atomically.

## 1.2 Domain Entities

> **Note on money types:** We use `float64` for prices throughout this project for simplicity. In production, **using floating point for money can cause rounding errors** (e.g., `0.1 + 0.2 = 0.30000000000000004`). The two common solutions are: (1) use integer cents internally (e.g., `1999` for $19.99), or (2) use a decimal library like `shopspring/decimal`. The database column `NUMERIC(10,2)` handles this correctly on the PostgreSQL side.

```go
// internal/domain/product.go
package domain

import (
    "errors"
    "fmt"
    "strings"
    "time"
    "unicode"
)

var (
    ErrProductNotFound   = errors.New("product not found")
    ErrProductOutOfStock = errors.New("product is out of stock")
    ErrInvalidProduct    = errors.New("invalid product")
    ErrInsufficientStock = errors.New("insufficient stock")
)

type Product struct {
    ID           int
    CategoryID   *int
    Name         string
    Slug         string
    Description  string
    Price        float64
    ComparePrice *float64
    Stock        int
    SKU          string
    WeightGrams  int
    IsActive     bool
    Images       []ProductImage
    Category     *Category
    CreatedAt    time.Time
    UpdatedAt    time.Time
}

type ProductImage struct {
    ID        int
    ProductID int
    URL       string
    AltText   string
    Position  int
    IsPrimary bool
    CreatedAt time.Time
}

func (p *Product) Validate() error {
    if strings.TrimSpace(p.Name) == "" {
        return fmt.Errorf("%w: name is required", ErrInvalidProduct)
    }
    if p.Price < 0 {
        return fmt.Errorf("%w: price cannot be negative", ErrInvalidProduct)
    }
    return nil
}

func (p *Product) IsOnSale() bool {
    return p.ComparePrice != nil && *p.ComparePrice > p.Price
}

func (p *Product) DiscountPercent() float64 {
    if !p.IsOnSale() {
        return 0
    }
    return ((*p.ComparePrice - p.Price) / *p.ComparePrice) * 100
}

func (p *Product) Reserve(qty int) error {
    if qty <= 0 {
        return fmt.Errorf("%w: quantity must be positive", ErrInvalidProduct)
    }
    if p.Stock < qty {
        return fmt.Errorf("%w: have %d, need %d", ErrInsufficientStock, p.Stock, qty)
    }
    p.Stock -= qty
    return nil
}

func (p *Product) PrimaryImage() *ProductImage {
    for i := range p.Images {
        if p.Images[i].IsPrimary {
            return &p.Images[i]
        }
    }
    if len(p.Images) > 0 {
        return &p.Images[0]
    }
    return nil
}

// GenerateSlug creates a URL-friendly slug from the product name
func GenerateSlug(name string) string {
    var b strings.Builder
    for _, r := range strings.ToLower(strings.TrimSpace(name)) {
        if unicode.IsLetter(r) || unicode.IsDigit(r) {
            b.WriteRune(r)
        } else if unicode.IsSpace(r) || r == '-' {
            b.WriteRune('-')
        }
    }
    return strings.Trim(b.String(), "-")
}
```

> **Warning — `float64` for money:** The domain entities above use `float64` for
> `Price`, `Subtotal`, `Total`, etc. This is a deliberate simplification for
> learning. In a production system, floating-point arithmetic causes rounding
> errors (e.g., `0.1 + 0.2 != 0.3`). The industry-standard approach is to store
> monetary values as **integer cents** (`int64`) and only convert to dollars at
> the display boundary. Alternatively, use a decimal library like
> `shopspring/decimal`. The PostgreSQL schema already uses `NUMERIC(10,2)` which
> is exact — the risk is in Go-side arithmetic between the database and the
> response. If you want to harden this project, replace all `float64` money
> fields with `int64` (cents) and divide by 100 only in the JSON serializer.

```go
import (

    ErrUserNotFound       = errors.New("user not found")

    RoleCustomer Role = "customer"
    RoleAdmin    Role = "admin"
)

type User struct {
    ID        int
    Email     string
    Password  string // hashed
    FirstName string
    LastName  string
    Role      Role
    CreatedAt time.Time
    UpdatedAt time.Time
}

func (u *User) Name() string {
    return u.FirstName + " " + u.LastName
}

// HashPassword replaces the plain-text password with its bcrypt hash.
func (u *User) HashPassword() error {
    hash, err := bcrypt.GenerateFromPassword([]byte(u.Password), bcrypt.DefaultCost)
    if err != nil {
        return err
    }
    u.Password = string(hash)
    return nil
}

// CheckPassword compares a plain-text candidate against the stored hash.
func (u *User) CheckPassword(candidate string) bool {
    return bcrypt.CompareHashAndPassword([]byte(u.Password), []byte(candidate)) == nil
}
```

```go
// internal/domain/order.go
package domain

import (
    "errors"
    "time"
)

var (
    ErrOrderNotFound            = errors.New("order not found")
    ErrInvalidStatusTransition  = errors.New("invalid order status transition")
)

type OrderStatus string

const (
    OrderStatusPending        OrderStatus = "pending"
    OrderStatusPaymentPending OrderStatus = "payment_pending"
    OrderStatusPaymentFailed  OrderStatus = "payment_failed"
    OrderStatusConfirmed      OrderStatus = "confirmed"
    OrderStatusProcessing     OrderStatus = "processing"
    OrderStatusShipped        OrderStatus = "shipped"
    OrderStatusDelivered      OrderStatus = "delivered"
    OrderStatusCancelled      OrderStatus = "cancelled"
    OrderStatusRefunded       OrderStatus = "refunded"
)

// validTransitions defines which status transitions are allowed.
var validTransitions = map[OrderStatus][]OrderStatus{
    OrderStatusPending:        {OrderStatusPaymentPending, OrderStatusCancelled},
    OrderStatusPaymentPending: {OrderStatusConfirmed, OrderStatusPaymentFailed, OrderStatusCancelled},
    OrderStatusPaymentFailed:  {OrderStatusPaymentPending, OrderStatusCancelled},
    OrderStatusConfirmed:      {OrderStatusProcessing, OrderStatusCancelled, OrderStatusRefunded},
    OrderStatusProcessing:     {OrderStatusShipped, OrderStatusCancelled, OrderStatusRefunded},
    OrderStatusShipped:        {OrderStatusDelivered, OrderStatusRefunded},
    OrderStatusDelivered:      {OrderStatusRefunded},
}

// CanTransitionTo returns true if moving from the current status to next is allowed.
func (s OrderStatus) CanTransitionTo(next OrderStatus) bool {
    for _, allowed := range validTransitions[s] {
        if allowed == next {
            return true
        }
    }
    return false
}
```

```go
// internal/domain/cart.go
package domain

import (
    "errors"
    "fmt"
)

var (
    ErrCartNotFound    = errors.New("cart not found")
    ErrCartItemInvalid = errors.New("invalid cart item")
)

type Cart struct {
    SessionID string
    UserID    *int     // nil for guest carts
    Items     []CartItem
}

type CartItem struct {
    ProductID int
    Product   *Product // hydrated for display
    Quantity  int
}

func (c *Cart) AddItem(productID, qty int) {
    for i := range c.Items {
        if c.Items[i].ProductID == productID {
            c.Items[i].Quantity += qty
            return
        }
    }
    c.Items = append(c.Items, CartItem{ProductID: productID, Quantity: qty})
}

func (c *Cart) RemoveItem(productID int) {
    for i, item := range c.Items {
        if item.ProductID == productID {
            c.Items = append(c.Items[:i], c.Items[i+1:]...)
            return
        }
    }
}

func (c *Cart) UpdateQuantity(productID, qty int) error {
    if qty < 0 {
        return fmt.Errorf("%w: quantity cannot be negative", ErrCartItemInvalid)
    }
    if qty == 0 {
        c.RemoveItem(productID)
        return nil
    }
    for i := range c.Items {
        if c.Items[i].ProductID == productID {
            c.Items[i].Quantity = qty
            return nil
        }
    }
    return ErrCartNotFound
}

func (c *Cart) Subtotal() float64 {
    var total float64
    for _, item := range c.Items {
        if item.Product != nil {
            total += item.Product.Price * float64(item.Quantity)
        }
    }
    return total
}

func (c *Cart) ItemCount() int {
    count := 0
    for _, item := range c.Items {
        count += item.Quantity
    }
    return count
}

func (c *Cart) IsEmpty() bool {
    return len(c.Items) == 0
}
```

## 1.3b Standard API Response Helpers

Every endpoint should return a consistent JSON envelope. This small package
eliminates boilerplate and guarantees uniform error shapes across the entire API.

```go
// pkg/response/response.go
package response

import (
    "net/http"

    "github.com/gin-gonic/gin"
)

// Response is the standard JSON envelope for all API responses.
type Response struct {
    Success bool   `json:"success"`
    Data    any    `json:"data,omitempty"`
    Error   string `json:"error,omitempty"`
    Meta    *Meta  `json:"meta,omitempty"`
}

// Meta carries pagination information.
type Meta struct {
    Page       int `json:"page"`
    Limit      int `json:"limit"`
    Total      int `json:"total"`
    TotalPages int `json:"total_pages"`
}

// OK sends a 200 response with data.
func OK(c *gin.Context, data any) {
    c.JSON(http.StatusOK, Response{Success: true, Data: data})
}

// Created sends a 201 response with the newly created resource.
func Created(c *gin.Context, data any) {
    c.JSON(http.StatusCreated, Response{Success: true, Data: data})
}

// BadRequest sends a 400 response with an error message.
func BadRequest(c *gin.Context, msg string) {
    c.JSON(http.StatusBadRequest, Response{Success: false, Error: msg})
}

// NotFound sends a 404 response with an error message.
func NotFound(c *gin.Context, msg string) {
    c.JSON(http.StatusNotFound, Response{Success: false, Error: msg})
}

// InternalError sends a generic 500 response.
// Never leak internal details — the caller logs the real error separately.
func InternalError(c *gin.Context) {
    c.JSON(http.StatusInternalServerError, Response{
        Success: false,
        Error:   "internal server error",
    })
}

// Paginated sends a 200 response with data and pagination metadata.
func Paginated(c *gin.Context, data any, page, limit, total int) {
    totalPages := total / limit
    if total%limit != 0 {
        totalPages++
    }
    c.JSON(http.StatusOK, Response{
        Success: true,
        Data:    data,
        Meta: &Meta{
            Page:       page,
            Limit:      limit,
            Total:      total,
            TotalPages: totalPages,
        },
    })
}
```

```go
    "fmt"
```

```go
type MinIOService struct {
    client   *minio.Client
    bucket   string
    endpoint string
    useSSL   bool
}

func NewMinIO(endpoint, accessKey, secretKey, bucket string, useSSL bool) (*MinIOService, error) {
    client, err := minio.New(endpoint, &minio.Options{
        Creds:  credentials.NewStaticV4(accessKey, secretKey, ""),
        Secure: useSSL,
    })
    if err != nil {
        return nil, fmt.Errorf("minio client: %w", err)
    }

    svc := &MinIOService{
        client:   client,
        bucket:   bucket,
        endpoint: endpoint,
        useSSL:   useSSL,
    }

    // Ensure bucket exists
    if err := svc.ensureBucket(context.Background()); err != nil {
        return nil, err
    }

    return svc, nil
}

func (s *MinIOService) ensureBucket(ctx context.Context) error {
    exists, err := s.client.BucketExists(ctx, s.bucket)
    if err != nil {
        return fmt.Errorf("check bucket: %w", err)
    }
    if exists {
        return nil
    }

    if err := s.client.MakeBucket(ctx, s.bucket, minio.MakeBucketOptions{}); err != nil {
        return fmt.Errorf("create bucket: %w", err)
    }

    // Set public read policy for product images
    policy := fmt.Sprintf(`{
        "Version":"2012-10-17",
        "Statement":[{
            "Effect":"Allow",
            "Principal":{"AWS":["*"]},
            "Action":["s3:GetObject"],
            "Resource":["arn:aws:s3:::%s/*"]
        }]
    }`, s.bucket)
    return s.client.SetBucketPolicy(ctx, s.bucket, policy)
}

func (s *MinIOService) Upload(ctx context.Context, key string, data []byte, contentType string) (string, error) {
    _, err := s.client.PutObject(ctx, s.bucket, key,
        bytes.NewReader(data), int64(len(data)),
        minio.PutObjectOptions{ContentType: contentType},
    )
    if err != nil {
        return "", fmt.Errorf("upload %s: %w", key, err)
    }
    return s.GetURL(key), nil
}

func (s *MinIOService) Delete(ctx context.Context, key string) error {
    return s.client.RemoveObject(ctx, s.bucket, key, minio.RemoveObjectOptions{})
}

func (s *MinIOService) GetURL(key string) string {
    scheme := "http"
    if s.useSSL {
        scheme = "https"
    }
    return fmt.Sprintf("%s://%s/%s/%s", scheme, s.endpoint, s.bucket, key)
}
```

## 2.3 Image Processing Before Upload

Always resize and compress images before storing them. A user's phone photo
is 8MB — you do not need that in your database.

```go
// pkg/storage/image.go
package storage

import (
    "bytes"
    "fmt"
    "image"
    _ "image/gif"
    "image/jpeg"
    _ "image/png"

    "github.com/disintegration/imaging"
)

const (
    MaxUploadBytes = 10 << 20  // 10 MB
    ProductImageW  = 800
    ProductImageH  = 800
    ThumbW         = 200
    ThumbH         = 200
    JPEGQuality    = 85
)

type ProcessedImage struct {
    Full      []byte
    Thumbnail []byte
}

// ProcessImage decodes, resizes to a max dimension, and re-encodes as JPEG.
func ProcessImage(data []byte) (*ProcessedImage, error) {
    img, _, err := image.Decode(bytes.NewReader(data))
    if err != nil {
        return nil, fmt.Errorf("decode image: %w", err)
    }

    // Resize maintaining aspect ratio — fit within ProductImageW x ProductImageH
    full := imaging.Fit(img, ProductImageW, ProductImageH, imaging.Lanczos)

    fullBytes, err := encodeJPEG(full)
    if err != nil {
        return nil, err
    }

    // Generate thumbnail
    thumb := imaging.Thumbnail(img, ThumbW, ThumbH, imaging.Lanczos)
    thumbBytes, err := encodeJPEG(thumb)
    if err != nil {
        return nil, err
    }

    return &ProcessedImage{Full: fullBytes, Thumbnail: thumbBytes}, nil
}

func encodeJPEG(img image.Image) ([]byte, error) {
    var buf bytes.Buffer
    if err := jpeg.Encode(&buf, img, &jpeg.Options{Quality: JPEGQuality}); err != nil {
        return nil, fmt.Errorf("encode JPEG: %w", err)
    }
    return buf.Bytes(), nil
}
```

```go
package http

import (
    "fmt"
    "net/http"
    "path/filepath"
    "strings"
    "time"

    "github.com/gin-gonic/gin"
    "github.com/google/uuid"
    "github.com/yourname/ecommerce/pkg/storage"
    "github.com/yourname/ecommerce/pkg/response"
)

func (h *ProductHandler) UploadImage(c *gin.Context) {
    productID, err := parseID(c, "id")
    if err != nil {
        response.BadRequest(c, "invalid product ID")
        return
    }

    // Verify product exists
    product, err := h.uc.GetByID(c.Request.Context(), productID)
    if err != nil {
        response.NotFound(c, "product not found")
        return
    }

    // Limit request body size before parsing
    c.Request.Body = http.MaxBytesReader(c.Writer, c.Request.Body, storage.MaxUploadBytes)

    // Parse multipart form
    if err := c.Request.ParseMultipartForm(storage.MaxUploadBytes); err != nil {
        response.BadRequest(c, "file too large or invalid form")
        return
    }

    file, header, err := c.Request.FormFile("image")
    if err != nil {
        response.BadRequest(c, "image file is required")
        return
    }
    defer file.Close()

    // Validate content type
    contentType := header.Header.Get("Content-Type")
    if !isAllowedImageType(contentType) {
        response.BadRequest(c, "only JPEG, PNG, and WebP images are allowed")
        return
    }

    // Read file data
    data := make([]byte, header.Size)
    if _, err := file.Read(data); err != nil {
        response.InternalError(c)
        return
    }

    // Process image (resize + compress)
    processed, err := storage.ProcessImage(data)
    if err != nil {
        response.BadRequest(c, "invalid image file")
        return
    }

    // Generate unique storage keys
    id := uuid.New().String()
    fullKey  := fmt.Sprintf("products/%d/%s.jpg", productID, id)
    thumbKey := fmt.Sprintf("products/%d/%s_thumb.jpg", productID, id)

    // Upload both versions
    altText := c.PostForm("alt_text")
    if altText == "" {
        altText = product.Name
    }

    img, err := h.uc.AddImage(c.Request.Context(), usecase.AddImageInput{
        ProductID:    productID,
        FullKey:      fullKey,
        ThumbKey:     thumbKey,
        FullData:     processed.Full,
        ThumbnailData: processed.Thumbnail,
        AltText:      altText,
        IsPrimary:    len(product.Images) == 0, // first image is primary
    })
    if err != nil {
        response.InternalError(c)
        return
    }

    response.Created(c, img)
}

func isAllowedImageType(ct string) bool {
    allowed := []string{"image/jpeg", "image/png", "image/webp"}
    for _, a := range allowed {
        if strings.HasPrefix(ct, a) {
            return true
        }
    }
    return false
}
```

# Week 3 — Shopping Cart

## 3.1 Cart Design Decisions

The cart lives in Redis, not PostgreSQL. Reasons:
- Carts are volatile — most are abandoned and never become orders
- They need sub-millisecond reads on every page load
- Redis TTL naturally expires abandoned carts

Guests get a cart session ID stored in a cookie. When they log in,
their guest cart merges into their user cart.

**Concurrency:** If the same user adds items from two browser tabs simultaneously,
the second write can silently overwrite the first. To prevent this, use Redis
`WATCH`/`MULTI` transactions: `WATCH` the cart key, read the current value,
modify it in-memory, then `MULTI`/`EXEC` the write. If another client changed the
key between `WATCH` and `EXEC`, the transaction aborts and you retry.

**Redis failure trade-off:** If Redis is temporarily unavailable, the cart is lost
but the rest of the application keeps working. This is an acceptable trade-off —
carts are not revenue until they become orders. The alternative (storing carts in
PostgreSQL) adds write load for data that is mostly thrown away.

**JSON vs Redis hashes:** We store the entire cart as a single JSON blob rather than
using Redis hash fields per item. JSON is simpler to reason about, easier to
deserialize into a Go struct, and a single `GET`/`SET` is faster than `HGETALL`
when carts are small (typically under 20 items). If your carts routinely exceeded
100 items, per-field hashes would reduce serialization overhead.

## 3.2 Cart Repository (Redis)

```go
// internal/repository/redis/cart.go
package redis

import (
    "context"
    "encoding/json"
    "errors"
    "fmt"
    "time"

    "github.com/redis/go-redis/v9"
    "github.com/yourname/ecommerce/internal/domain"
)

const cartTTL = 7 * 24 * time.Hour // 7 days

type CartRepository struct {
    client *redis.Client
}

func NewCartRepository(client *redis.Client) *CartRepository {
    return &CartRepository{client: client}
}

func (r *CartRepository) key(sessionID string) string {
    return fmt.Sprintf("cart:%s", sessionID)
}

func (r *CartRepository) Get(ctx context.Context, sessionID string) (*domain.Cart, error) {
    data, err := r.client.Get(ctx, r.key(sessionID)).Bytes()
    if err == redis.Nil {
        // Return an empty cart — not an error
        return &domain.Cart{SessionID: sessionID}, nil
    }
    if err != nil {
        return nil, fmt.Errorf("cart get: %w", err)
    }

    var cart domain.Cart
    if err := json.Unmarshal(data, &cart); err != nil {
        return nil, fmt.Errorf("cart decode: %w", err)
    }
    return &cart, nil
}

func (r *CartRepository) Save(ctx context.Context, cart *domain.Cart) error {
    data, err := json.Marshal(cart)
    if err != nil {
        return err
    }
    return r.client.Set(ctx, r.key(cart.SessionID), data, cartTTL).Err()
}

func (r *CartRepository) Delete(ctx context.Context, sessionID string) error {
    return r.client.Del(ctx, r.key(sessionID)).Err()
}

// MergeGuestCart merges guest cart items into the user's cart.
// After merging, the guest cart is deleted.
func (r *CartRepository) MergeGuestCart(ctx context.Context, guestSessionID string, userID int) error {
    guestCart, err := r.Get(ctx, guestSessionID)
    if err != nil || len(guestCart.Items) == 0 {
        return err
    }

    userSessionID := fmt.Sprintf("user:%d", userID)
    userCart, err := r.Get(ctx, userSessionID)
    if err != nil {
        return err
    }

    // Merge: add guest items to user cart, summing quantities for duplicates
    for _, guestItem := range guestCart.Items {
        userCart.AddItem(guestItem.ProductID, guestItem.Quantity)
    }
    userCart.SessionID = userSessionID
    userCart.UserID = &userID

    if err := r.Save(ctx, userCart); err != nil {
        return err
    }

    return r.Delete(ctx, guestSessionID)
}
```

## 3.3 Cart Use Case

```go
// internal/usecase/cart.go
package usecase

import (
    "context"
    "fmt"

    "github.com/yourname/ecommerce/internal/domain"
)

type CartUseCase struct {
    cartRepo    domain.CartRepository
    productRepo domain.ProductRepository
}

func NewCartUseCase(cartRepo domain.CartRepository, productRepo domain.ProductRepository) *CartUseCase {
    return &CartUseCase{cartRepo: cartRepo, productRepo: productRepo}
}

// GetCart returns the cart with hydrated product data for display.
func (uc *CartUseCase) GetCart(ctx context.Context, sessionID string) (*domain.Cart, error) {
    cart, err := uc.cartRepo.Get(ctx, sessionID)
    if err != nil {
        return nil, err
    }

    // Hydrate products — fetch each product for display info
    for i := range cart.Items {
        p, err := uc.productRepo.GetByID(ctx, cart.Items[i].ProductID)
        if err != nil {
            // Product was deleted — remove from cart
            cart.RemoveItem(cart.Items[i].ProductID)
            continue
        }
        cart.Items[i].Product = p
    }

    // Save in case any items were removed
    _ = uc.cartRepo.Save(ctx, cart)
    return cart, nil
}

func (uc *CartUseCase) AddItem(ctx context.Context, sessionID string, productID, qty int) (*domain.Cart, error) {
    // Validate product exists and has stock
    product, err := uc.productRepo.GetByID(ctx, productID)
    if err != nil {
        return nil, fmt.Errorf("add to cart: %w", err)
    }
    if !product.IsActive {
        return nil, fmt.Errorf("add to cart: product is not available")
    }
    if product.Stock < qty {
        return nil, fmt.Errorf("add to cart: %w", domain.ErrInsufficientStock)
    }

    cart, err := uc.cartRepo.Get(ctx, sessionID)
    if err != nil {
        return nil, err
    }

    cart.AddItem(productID, qty)

    if err := uc.cartRepo.Save(ctx, cart); err != nil {
        return nil, err
    }

    // Hydrate for response
    cart.Items[len(cart.Items)-1].Product = product
    return cart, nil
}

func (uc *CartUseCase) UpdateItem(ctx context.Context, sessionID string, productID, qty int) (*domain.Cart, error) {
```

**Checkout Sessions** are Stripe's recommended integration path. Instead of
collecting card details on your server (which triggers PCI compliance requirements),
you redirect the customer to a Stripe-hosted payment page. Stripe handles card
input, 3D Secure, Apple Pay, and Google Pay — then redirects back to your
`success_url` or `cancel_url`. Your server never touches card numbers.

**Local webhook testing:** During development, Stripe cannot reach `localhost`.
Install the Stripe CLI and run:
```bash
stripe listen --forward-to localhost:8080/api/v1/webhooks/stripe
```
This creates a temporary tunnel. The CLI prints a webhook signing secret
(`whsec_...`) — use that as your `STRIPE_WEBHOOK_SECRET` in `.env`.

**Test card numbers:** Use `4242 4242 4242 4242` with any future expiry and any
CVC to simulate a successful payment. Use `4000 0000 0000 0002` to simulate a
decline. See the Stripe docs for the full list of test cards.

```go
// pkg/stripe/stripe.go
package stripe

import (
    "fmt"

    "github.com/stripe/stripe-go/v76"
    "github.com/stripe/stripe-go/v76/checkout/session"
    "github.com/stripe/stripe-go/v76/webhook"
)

type Client struct {
    secretKey      string
    webhookSecret  string
    successURL     string
    cancelURL      string
}

func NewClient(secretKey, webhookSecret, baseURL string) *Client {
    stripe.Key = secretKey
    return &Client{
        secretKey:     secretKey,
        webhookSecret: webhookSecret,
        successURL:    baseURL + "/success?session_id={CHECKOUT_SESSION_ID}",
        cancelURL:     baseURL + "/cancel",
    }
}

type LineItem struct {
    Name      string
    ImageURL  string
    UnitPrice int64  // in cents
    Quantity  int64
}

type CheckoutSession struct {
    ID  string
    URL string
}

func (c *Client) CreateCheckoutSession(orderID int, items []LineItem, email string) (*CheckoutSession, error) {
    params := &stripe.CheckoutSessionParams{
        Mode:              stripe.String(string(stripe.CheckoutSessionModePayment)),
        SuccessURL:        stripe.String(c.successURL),
        CancelURL:         stripe.String(c.cancelURL),
        CustomerEmail:     stripe.String(email),
        LineItems:         make([]*stripe.CheckoutSessionLineItemParams, len(items)),
    }

    // Attach order ID so we can match it in the webhook
    params.AddMetadata("order_id", fmt.Sprintf("%d", orderID))

    for i, item := range items {
        imgURLs := []*string{}
        if item.ImageURL != "" {
            imgURLs = append(imgURLs, stripe.String(item.ImageURL))
        }

        params.LineItems[i] = &stripe.CheckoutSessionLineItemParams{
            PriceData: &stripe.CheckoutSessionLineItemPriceDataParams{
                Currency: stripe.String("usd"),
                UnitAmount: stripe.Int64(item.UnitPrice),
                ProductData: &stripe.CheckoutSessionLineItemPriceDataProductDataParams{
                    Name:   stripe.String(item.Name),
                    Images: imgURLs,
                },
            },
            Quantity: stripe.Int64(item.Quantity),
        }
    }

    s, err := session.New(params)
    if err != nil {
        return nil, fmt.Errorf("create checkout session: %w", err)
    }

    return &CheckoutSession{ID: s.ID, URL: s.URL}, nil
}

// ParseWebhookEvent validates the Stripe signature and returns the event.
func (c *Client) ParseWebhookEvent(payload []byte, sig string) (*stripe.Event, error) {
    event, err := webhook.ConstructEvent(payload, sig, c.webhookSecret)
    if err != nil {
        return nil, fmt.Errorf("webhook signature verification failed: %w", err)
    }
    return &event, nil
}
```

## 4.4 Checkout Use Case

```go
// internal/usecase/payment.go
package usecase

import (
    "context"
    "fmt"

    "github.com/yourname/ecommerce/internal/domain"
    "github.com/yourname/ecommerce/pkg/stripe"
)

type PaymentUseCase struct {
    orderRepo   domain.OrderRepository
    productRepo domain.ProductRepository
    cartRepo    domain.CartRepository
    paymentRepo domain.PaymentRepository
    userRepo    domain.UserRepository
    stripe      *stripe.Client
    jobQueue    JobQueue
}

type JobQueue interface {
    Enqueue(ctx context.Context, jobType string, payload any) error
}

func (uc *PaymentUseCase) CreateCheckout(ctx context.Context, userID int, sessionID string) (*stripe.CheckoutSession, error) {
    // 1. Get and validate cart
    cart, err := uc.cartRepo.Get(ctx, sessionID)
    if err != nil || cart.IsEmpty() {
        return nil, fmt.Errorf("checkout: cart is empty")
    }

    // 2. Get user for email
    user, err := uc.userRepo.GetByID(ctx, userID)
    if err != nil {
        return nil, err
    }

    // 3. Validate all items and calculate total
    var items []stripe.LineItem
    var orderItems []domain.OrderItem
    var subtotal float64

    for _, cartItem := range cart.Items {
        product, err := uc.productRepo.GetByID(ctx, cartItem.ProductID)
        if err != nil {
            return nil, fmt.Errorf("product %d: %w", cartItem.ProductID, err)
        }
        if !product.IsActive {
            return nil, fmt.Errorf("product %q is no longer available", product.Name)
        }
        if product.Stock < cartItem.Quantity {
            return nil, fmt.Errorf("product %q: %w", product.Name, domain.ErrInsufficientStock)
        }

        lineTotal := product.Price * float64(cartItem.Quantity)
        subtotal += lineTotal

        imgURL := ""
        if img := product.PrimaryImage(); img != nil {
            imgURL = img.URL
        }

        items = append(items, stripe.LineItem{
            Name:      product.Name,
            ImageURL:  imgURL,
            UnitPrice: int64(product.Price * 100), // convert to cents
            Quantity:  int64(cartItem.Quantity),
        })

        orderItems = append(orderItems, domain.OrderItem{
            ProductID:   product.ID,
            ProductName: product.Name,
            ProductSKU:  product.SKU,
            Quantity:    cartItem.Quantity,
            UnitPrice:   product.Price,
            TotalPrice:  lineTotal,
        })
    }

    // 4. Create pending order in database
    order, err := uc.orderRepo.Create(ctx, &domain.Order{
        UserID:   userID,
        Items:    orderItems,
        Subtotal: subtotal,
        Total:    subtotal, // add tax/shipping logic here
        Status:   domain.OrderStatusPaymentPending,
    })
    if err != nil {
        return nil, fmt.Errorf("create order: %w", err)
    }

    // 5. Create Stripe checkout session
    stripeSession, err := uc.stripe.CreateCheckoutSession(order.ID, items, user.Email)
    if err != nil {
        // Roll back — mark order as failed
        _ = uc.orderRepo.UpdateStatus(ctx, order.ID, domain.OrderStatusPaymentFailed)
        return nil, fmt.Errorf("stripe session: %w", err)
    }

    // 6. Attach Stripe session ID to order
    _ = uc.orderRepo.AttachPayment(ctx, order.ID, stripeSession.ID, "")

    return stripeSession, nil
}

// HandleWebhook processes incoming Stripe webhook events.
func (uc *PaymentUseCase) HandleWebhook(ctx context.Context, payload []byte, sig string) error {
    event, err := uc.stripe.ParseWebhookEvent(payload, sig)
    if err != nil {
        return err
    }

    switch event.Type {
    case "checkout.session.completed":
        return uc.handleCheckoutCompleted(ctx, event)
    case "checkout.session.expired":
        return uc.handleCheckoutExpired(ctx, event)
    case "payment_intent.payment_failed":
        return uc.handlePaymentFailed(ctx, event)
    }

    return nil // ignore unknown events
}

func (uc *PaymentUseCase) handleCheckoutCompleted(ctx context.Context, event *stripe.Event) error {
    var s stripe.CheckoutSession
    if err := json.Unmarshal(event.Data.Raw, &s); err != nil {
        return err
    }

    orderIDStr := s.Metadata["order_id"]
    orderID, _ := strconv.Atoi(orderIDStr)

    // Get the order
    order, err := uc.orderRepo.GetByID(ctx, orderID)
    if err != nil {
        return err
    }

    // Mark as confirmed
    if err := uc.orderRepo.UpdateStatus(ctx, orderID, domain.OrderStatusConfirmed); err != nil {
        return err
    }

    // Deduct stock for each item
    for _, item := range order.Items {
        if err := uc.productRepo.UpdateStock(ctx, item.ProductID, -item.Quantity); err != nil {
            slog.Error("failed to deduct stock", "product_id", item.ProductID, "err", err)
            // Don't fail the webhook — stock can be manually corrected
        }
    }

    // Record payment
    _ = uc.paymentRepo.Create(ctx, &domain.Payment{
        OrderID:        orderID,
        StripePaymentID: s.PaymentIntent,
        Amount:         float64(s.AmountTotal) / 100,
        Status:         domain.PaymentStatusSucceeded,
    })

    // Clear the cart
    _ = uc.cartRepo.Delete(ctx, fmt.Sprintf("user:%d", order.UserID))

    // Enqueue notifications
    _ = uc.jobQueue.Enqueue(ctx, "email.order_confirmed", OrderConfirmedPayload{
        OrderID: orderID,
        UserID:  order.UserID,
    })
    _ = uc.jobQueue.Enqueue(ctx, "pdf.invoice", InvoicePayload{
        OrderID: orderID,
    })

    return nil
}

// handleCheckoutExpired is called when a Stripe Checkout Session expires
// (default: 24 hours). The customer never completed payment, so we cancel
// the pending order and restore any reserved stock.
func (uc *PaymentUseCase) handleCheckoutExpired(ctx context.Context, event *stripe.Event) error {
    var s stripe.CheckoutSession
    if err := json.Unmarshal(event.Data.Raw, &s); err != nil {
        return err
    }

    orderIDStr := s.Metadata["order_id"]
    orderID, _ := strconv.Atoi(orderIDStr)

    order, err := uc.orderRepo.GetByID(ctx, orderID)
    if err != nil {
        return fmt.Errorf("get expired order: %w", err)
    }

    // Only cancel if the order is still in a payment-pending state
    if order.Status != domain.OrderStatusPaymentPending {
        return nil
    }

    if err := uc.orderRepo.UpdateStatus(ctx, orderID, domain.OrderStatusCancelled); err != nil {
        return fmt.Errorf("cancel expired order: %w", err)
    }

    // Restore reserved stock for each item
    for _, item := range order.Items {
        if err := uc.productRepo.UpdateStock(ctx, item.ProductID, item.Quantity); err != nil {
            slog.Error("failed to restore stock after expiry",
                "product_id", item.ProductID, "qty", item.Quantity, "err", err)
        }
    }

    return nil
}

// handlePaymentFailed is called when a PaymentIntent fails (e.g. card declined).
// We mark the order as payment_failed so the customer can retry.
func (uc *PaymentUseCase) handlePaymentFailed(ctx context.Context, event *stripe.Event) error {
    var pi stripe.PaymentIntent
    if err := json.Unmarshal(event.Data.Raw, &pi); err != nil {
        return err
    }

    orderIDStr := pi.Metadata["order_id"]
    orderID, _ := strconv.Atoi(orderIDStr)

    if err := uc.orderRepo.UpdateStatus(ctx, orderID, domain.OrderStatusPaymentFailed); err != nil {
        return fmt.Errorf("mark order payment failed: %w", err)
    }

    // Record the failed payment for audit
    _ = uc.paymentRepo.Create(ctx, &domain.Payment{
        OrderID:         orderID,
        StripePaymentID: pi.ID,
        Amount:          float64(pi.Amount) / 100,
        Status:          domain.PaymentStatusFailed,
    })

    return nil
}
```

```go
// internal/delivery/http/payment.go
package http

import (
    "io"
    "net/http"

    "github.com/gin-gonic/gin"
    "github.com/yourname/ecommerce/pkg/response"
)

type PaymentHandler struct {
    uc *usecase.PaymentUseCase
}

func (h *PaymentHandler) CreateCheckoutSession(c *gin.Context) {
    userID := mustGetUserID(c)

    sessionID := fmt.Sprintf("user:%d", userID)
    stripeSession, err := h.uc.CreateCheckout(c.Request.Context(), userID, sessionID)
    if err != nil {
        switch {
        case strings.Contains(err.Error(), "cart is empty"):
            response.BadRequest(c, "cart is empty")
        case errors.Is(err, domain.ErrInsufficientStock):
            response.BadRequest(c, err.Error())
        default:
            slog.Error("checkout error", "user_id", userID, "err", err)
            response.InternalError(c)
        }
        return
    }

    response.OK(c, gin.H{
        "session_id":  stripeSession.ID,
        "checkout_url": stripeSession.URL,
    })
}

// Webhook must NOT use body-binding middleware — Stripe requires the raw body for signature verification
func (h *PaymentHandler) StripeWebhook(c *gin.Context) {
    // Read raw body before any middleware touches it
    payload, err := io.ReadAll(io.LimitReader(c.Request.Body, 65536))
    if err != nil {
        c.Status(http.StatusBadRequest)
        return
    }

    sig := c.GetHeader("Stripe-Signature")
    if sig == "" {
        c.Status(http.StatusBadRequest)
        return
    }

    if err := h.uc.HandleWebhook(c.Request.Context(), payload, sig); err != nil {
        slog.Error("webhook error", "err", err)
        // Return 400 so Stripe retries
        c.JSON(http.StatusBadRequest, gin.H{"error": err.Error()})
        return
    }

    c.Status(http.StatusOK)
}
```

```go
import (
    "context"

    "fmt"
    "net/smtp"
    "strings"
)

type SMTPSender struct {
    host     string
    port     int
    username string
    password string
    from     string
}

func NewSMTPSender(host string, port int, username, password, from string) *SMTPSender {
    return &SMTPSender{host: host, port: port, username: username, password: password, from: from}
}

func (s *SMTPSender) Send(ctx context.Context, msg Message) error {
    auth := smtp.PlainAuth("", s.username, s.password, s.host)

    headers := fmt.Sprintf(
        "From: %s\r\nTo: %s\r\nSubject: %s\r\nMIME-Version: 1.0\r\nContent-Type: text/html; charset=utf-8\r\n\r\n",
        s.from,
        strings.Join(msg.To, ", "),
        msg.Subject,
    )

    body := headers + msg.HTML
    addr := fmt.Sprintf("%s:%d", s.host, s.port)

    return smtp.SendMail(addr, auth, s.from, msg.To, []byte(body))
}
```

```go
type InvoiceData struct {

    CustomerEmail string

    Subtotal     float64
    Tax          float64
    Total        float64

    Quantity  int

    Total     float64

func GenerateInvoice(data InvoiceData) ([]byte, error) {
    cfg := config.NewBuilder().
        WithPageNumber("Page {current} of {total}", props.Right).
        Build()

    m := maroto.New(cfg)

    // Header
    m.AddRow(20,
        col.New(12).Add(
            text.New("INVOICE", props.Text{Size: 24, Style: fontstyle.Bold}),
        ),
    )

    m.AddRow(10,
        col.New(6).Add(text.New(fmt.Sprintf("Order #%d", data.OrderID))),
        col.New(6).Add(text.New(data.CreatedAt.Format("January 2, 2006"), props.Text{Align: align.Right})),
    )

    // Bill To
    m.AddRow(8, col.New(12).Add(text.New("Bill To:", props.Text{Style: fontstyle.Bold})))
    m.AddRow(6, col.New(12).Add(text.New(data.CustomerName)))
    m.AddRow(6, col.New(12).Add(text.New(data.CustomerEmail)))

    m.AddRow(4) // spacer

    // Table header
    m.AddRow(8,
        col.New(6).Add(text.New("Item", props.Text{Style: fontstyle.Bold})),
        col.New(2).Add(text.New("Qty",   props.Text{Style: fontstyle.Bold, Align: align.Center})),
        col.New(2).Add(text.New("Price", props.Text{Style: fontstyle.Bold, Align: align.Right})),
        col.New(2).Add(text.New("Total", props.Text{Style: fontstyle.Bold, Align: align.Right})),
    )

    // Items
    for _, item := range data.Items {
        m.AddRow(7,
            col.New(6).Add(text.New(item.Name)),
            col.New(2).Add(text.New(fmt.Sprintf("%d", item.Quantity), props.Text{Align: align.Center})),
            col.New(2).Add(text.New(fmt.Sprintf("$%.2f", item.UnitPrice), props.Text{Align: align.Right})),
            col.New(2).Add(text.New(fmt.Sprintf("$%.2f", item.Total), props.Text{Align: align.Right})),
        )
    }

    // Totals
    m.AddRow(4) // spacer
    m.AddRow(7,
        col.New(10).Add(text.New("Subtotal", props.Text{Align: align.Right})),
        col.New(2).Add(text.New(fmt.Sprintf("$%.2f", data.Subtotal), props.Text{Align: align.Right})),
    )
    m.AddRow(7,
        col.New(10).Add(text.New("Tax", props.Text{Align: align.Right})),
        col.New(2).Add(text.New(fmt.Sprintf("$%.2f", data.Tax), props.Text{Align: align.Right})),
    )
    m.AddRow(9,
        col.New(10).Add(text.New("Total", props.Text{Align: align.Right, Style: fontstyle.Bold, Size: 14})),
        col.New(2).Add(text.New(fmt.Sprintf("$%.2f", data.Total), props.Text{Align: align.Right, Style: fontstyle.Bold, Size: 14})),
    )

    doc, err := m.Generate()
    if err != nil {
        return nil, fmt.Errorf("generate invoice PDF: %w", err)
    }

    var buf bytes.Buffer
    if err := doc.Save(&buf); err != nil {
        return nil, err
    }

    return buf.Bytes(), nil
}
```

## 6.3 Inventory Management

```go
// internal/usecase/inventory.go
package usecase

type InventoryUseCase struct {
    productRepo domain.ProductRepository
    jobQueue    JobQueue
}

type StockAdjustment struct {
    ProductID int
    Delta     int    // positive = add stock, negative = remove
    Reason    string // "restock", "damage", "correction"
}

func (uc *InventoryUseCase) AdjustStock(ctx context.Context, adj StockAdjustment) (*domain.Product, error) {
    product, err := uc.productRepo.GetByID(ctx, adj.ProductID)
    if err != nil {
        return nil, err
    }

    newStock := product.Stock + adj.Delta
    if newStock < 0 {
        return nil, fmt.Errorf("adjustment would result in negative stock: have %d, adjusting by %d",
            product.Stock, adj.Delta)
    }

    if err := uc.productRepo.UpdateStock(ctx, adj.ProductID, adj.Delta); err != nil {
        return nil, fmt.Errorf("adjust stock: %w", err)
    }

    // Alert if stock is now low
    if newStock > 0 && newStock <= 5 {
        _ = uc.jobQueue.Enqueue(ctx, "alert.low_stock", LowStockPayload{
            ProductID: adj.ProductID,
            Name:      product.Name,
            Stock:     newStock,
        })
    }

    product.Stock = newStock
    return product, nil
}

// BulkStockUpdate updates stock for multiple products from a CSV import
func (uc *InventoryUseCase) BulkStockUpdate(ctx context.Context, updates []StockAdjustment) (int, []error) {
    var errs []error
    updated := 0

    for _, update := range updates {
        if _, err := uc.AdjustStock(ctx, update); err != nil {
            errs = append(errs, fmt.Errorf("product %d: %w", update.ProductID, err))
            continue
        }
        updated++
    }

    return updated, errs
}
```

```go
func PrometheusMiddleware() gin.HandlerFunc {
    return func(c *gin.Context) {
        start := time.Now()

        c.Next()

        duration := time.Since(start).Seconds()
        status   := strconv.Itoa(c.Writer.Status())
        path     := c.FullPath() // pattern path, not actual URL (avoids cardinality explosion)

        HTTPRequestsTotal.WithLabelValues(c.Request.Method, path, status).Inc()
        HTTPRequestDuration.WithLabelValues(c.Request.Method, path).Observe(duration)
    }
```

```go
func InitTracer(ctx context.Context, serviceName, endpoint string) (func(context.Context) error, error) {
    exporter, err := otlptracegrpc.New(ctx,
        otlptracegrpc.WithEndpoint(endpoint),
        otlptracegrpc.WithInsecure(),

    if err != nil {
        return nil, err

        trace.WithBatcher(exporter),
        trace.WithResource(resource.NewWithAttributes(
            semconv.SchemaURL,
            semconv.ServiceName(serviceName),
        )),
    )

    otel.SetTracerProvider(tp)
    return tp.Shutdown, nil
}
```

```go
// middleware: attach trace ID to logger
func LoggingMiddleware(logger *slog.Logger) gin.HandlerFunc {
    return func(c *gin.Context) {
        requestID := c.GetHeader("X-Request-ID")
        if requestID == "" {
            requestID = uuid.New().String()
        }

        // Get trace ID from OpenTelemetry context
        spanCtx := trace.SpanFromContext(c.Request.Context()).SpanContext()
        traceID := spanCtx.TraceID().String()

        // Create a logger with request context
        reqLogger := logger.With(
            "request_id", requestID,
            "trace_id", traceID,
            "method", c.Request.Method,
            "path", c.Request.URL.Path,
        )

        // Store in Gin context for use in handlers
        c.Set("logger", reqLogger)
        c.Header("X-Request-ID", requestID)

        rw := &responseWriter{ResponseWriter: c.Writer}
        start := time.Now()

        c.Next()

        reqLogger.Info("request completed",
            "status", rw.status,
            "latency_ms", time.Since(start).Milliseconds(),
            "bytes", rw.size,
        )
    }
}
```

```go
func SecurityHeaders() gin.HandlerFunc {
    return func(c *gin.Context) {
        // Prevent browsers from MIME sniffing
        c.Header("X-Content-Type-Options", "nosniff")
        // Prevent clickjacking
        c.Header("X-Frame-Options", "DENY")

        // Enforce HTTPS for 1 year

        // Strict Content Security Policy
        c.Header("Content-Security-Policy",
            "default-src 'self'; img-src 'self' data: https:; script-src 'self'")
        // Don't send Referrer to other origins
        c.Header("Referrer-Policy", "strict-origin-when-cross-origin")
        // Limit what browser features are allowed
        c.Header("Permissions-Policy", "camera=(), microphone=(), geolocation=()")

        c.Next()
    }
}
```

## 8.2 Input Sanitization

> **Output encoding vs input sanitization:** The `html.EscapeString` call below is
> technically *output encoding* applied at input time. This is a pragmatic shortcut
> for an API that only serves JSON — the escaped strings are stored in the database
> and returned as-is. If you later render these values in server-side HTML templates,
> Go's `html/template` package auto-escapes by default, so double-encoding is possible.
> The general rule: sanitize at the boundary closest to where damage can happen
> (SQL injection → parameterized queries; XSS → output encoding in templates;
> stored data → strip or escape on the way in when you know the output context).

```go
// pkg/sanitize/sanitize.go
package sanitize

import (
    "html"
    "strings"
    "unicode/utf8"
)

// Text strips HTML and trims whitespace — use for user-provided text fields
func Text(s string) string {
    s = strings.TrimSpace(s)
    s = html.EscapeString(s)  // escape <, >, &, ", '
    return s
}

// Truncate limits string length in runes (not bytes)
func Truncate(s string, max int) string {
    if utf8.RuneCountInString(s) <= max {
        return s
    }
    runes := []rune(s)
    return string(runes[:max])
}

// Use in use cases:
func (uc *ProductUseCase) Create(ctx context.Context, input CreateProductInput) (*domain.Product, error) {
    input.Name        = sanitize.Truncate(sanitize.Text(input.Name), 255)
    input.Description = sanitize.Truncate(sanitize.Text(input.Description), 5000)
    // ...
}
```

## 8.3 SQL Injection Prevention

You are already protected if you use parameterized queries (`$1`, `$2`).
Audit your codebase for any string formatting in SQL:

```go
// VULNERABLE — never do this
query := fmt.Sprintf("SELECT * FROM products WHERE name = '%s'", name)

// SAFE — always use parameters
row := db.QueryRowContext(ctx,
    "SELECT * FROM products WHERE name = $1", name)

// SAFE — dynamic WHERE with parameters (the correct approach)
where := []string{}
args  := []any{}
i := 1
if name != "" {
    where = append(where, fmt.Sprintf("name ILIKE $%d", i))
    args  = append(args, "%"+name+"%")
    i++
}
query := "SELECT * FROM products"
if len(where) > 0 {
    query += " WHERE " + strings.Join(where, " AND ")
}
```

```yaml
name: Build and Deploy

          go-version: "1.22"

          REDIS_URL: redis://localhost:6379/0
        run: |
          go test -race -coverprofile=coverage.out ./...
          go tool cover -func=coverage.out

      - name: Check coverage
        run: |
          COVERAGE=$(go tool cover -func=coverage.out | grep total | awk '{print $3}' | sed 's/%//')
          echo "Coverage: $COVERAGE%"
          if (( $(echo "$COVERAGE < 70" | bc -l) )); then
            echo "Coverage below 70%"
            exit 1
          fi

    needs: test

      - uses: actions/checkout@v4

          docker push ${{ secrets.REGISTRY }}/ecommerce-worker:${{ github.sha }}

  deploy:
    needs: build-and-push
    runs-on: ubuntu-latest
    steps:
      - name: Deploy to production
        uses: appleboy/ssh-action@master
        with:
          host: ${{ secrets.PROD_HOST }}
          username: ${{ secrets.PROD_USER }}
          key: ${{ secrets.PROD_SSH_KEY }}
          script: |
            cd /opt/ecommerce
            export IMAGE_TAG=${{ github.sha }}
            docker compose -f docker-compose.prod.yml pull
            docker compose -f docker-compose.prod.yml up -d
            docker compose -f docker-compose.prod.yml run --rm migrate
```
