# Minggu 1 — Bekerja dengan File & OS

## 1.1 Package `os`

Package `os` memberikan kamu akses ke sistem operasi: file, environment variable, argumen command-line, proses, dan lainnya.

### Environment Variable

```go
package main

import (
    "fmt"
    "os"
)

func main() {
    // Read an environment variable
    home := os.Getenv("HOME")
    fmt.Println("Home:", home)

    // Read with a default fallback
    port := os.Getenv("PORT")
    if port == "" {
        port = "8080"
    }
    fmt.Println("Port:", port)

    // Set an environment variable (current process only)
    os.Setenv("APP_ENV", "development")

    // Get all environment variables
    for _, env := range os.Environ() {
        fmt.Println(env) // "KEY=VALUE"
    }

    // Remove an environment variable
    os.Unsetenv("APP_ENV")
}
```

### Argumen Command-Line

```go
// os.Args[0] is the program name
// os.Args[1:] are the arguments

func main() {
    args := os.Args
    fmt.Println("Program:", args[0])

    if len(args) < 2 {
        fmt.Println("Usage: program <name>")
        os.Exit(1) // exit with non-zero code signals failure
    }

    fmt.Println("Hello,", args[1])
}
```

```bash
go run main.go Alice
# Program: /tmp/go-build.../exe/main
# Hello, Alice
```

### Bekerja dengan Filesystem

```go
import (
    "fmt"
    "os"
)

func main() {
    // Get current working directory
    dir, err := os.Getwd()
    if err != nil {
        fmt.Println("Error:", err)
        return
    }
    fmt.Println("CWD:", dir)

    // Create a directory
    err = os.Mkdir("testdir", 0755)
    if err != nil {
        fmt.Println("Error:", err)
    }

    // Create nested directories (like mkdir -p)
    err = os.MkdirAll("testdir/sub/deep", 0755)
    if err != nil {
        fmt.Println("Error:", err)
    }

    // Remove a file or empty directory
    os.Remove("testdir/file.txt")

    // Remove a directory and all its contents (like rm -rf)
    os.RemoveAll("testdir")

    // Rename or move a file
    os.Rename("old.txt", "new.txt")

    // Get file info
    info, err := os.Stat("main.go")
    if err != nil {
        if os.IsNotExist(err) {
            fmt.Println("file does not exist")
        }
        return
    }
    fmt.Println("Name:", info.Name())
    fmt.Println("Size:", info.Size(), "bytes")
    fmt.Println("Modified:", info.ModTime())
    fmt.Println("Is dir:", info.IsDir())
}
```

## 1.2 Membaca & Menulis File

### Membaca Seluruh File

```go
import (
    "fmt"
    "os"
)

func main() {
    // Read entire file into memory — fine for small files
    data, err := os.ReadFile("hello.txt")
    if err != nil {
        fmt.Println("Error reading file:", err)
        return
    }
    fmt.Println(string(data))
}
```

### Menulis Seluruh File

```go
func main() {
    content := []byte("Hello, Go!\nThis is a file.\n")

    // Write file — creates or truncates
    // 0644 = owner read/write, group read, others read
    err := os.WriteFile("output.txt", content, 0644)
    if err != nil {
        fmt.Println("Error writing file:", err)
        return
    }
    fmt.Println("File written successfully")
}
```

### Membuka File untuk Kontrol Lebih Lanjut

```go
import (
    "fmt"
    "os"
)

func main() {
    // Open for reading
    f, err := os.Open("data.txt")
    if err != nil {
        fmt.Println("Error:", err)
        return
    }
    defer f.Close() // always close

    // Open with flags (create, write, append)
    f2, err := os.OpenFile("log.txt",
        os.O_CREATE|os.O_WRONLY|os.O_APPEND,
        0644,
    )
    if err != nil {
        fmt.Println("Error:", err)
        return
    }
    defer f2.Close()

    f2.WriteString("Log entry\n")
}
```

**Flag yang umum digunakan:**
- `os.O_RDONLY` — hanya baca
- `os.O_WRONLY` — hanya tulis
- `os.O_RDWR` — baca dan tulis
- `os.O_CREATE` — buat jika belum ada
- `os.O_TRUNC` — kosongkan isi file saat dibuka
- `os.O_APPEND` — tambahkan di akhir

## 1.3 Package `io` — Membaca & Menulis Stream

Package `io` mendefinisikan interface inti `Reader` dan `Writer` yang digunakan oleh semua hal di Go.

```go
type Reader interface {
    Read(p []byte) (n int, err error)
}

type Writer interface {
    Write(p []byte) (n int, err error)
}
```

Tipe apa pun yang mengimplementasikan `Read` adalah sebuah `Reader`. File, HTTP response body, string, byte buffer — semuanya adalah reader. Inilah mengapa fungsi yang sama bisa bekerja dengan semuanya.

### io.Copy — Stream dari Reader ke Writer

```go
import (
    "io"
    "os"
)

func main() {
    // Copy a file
    src, _ := os.Open("source.txt")
    defer src.Close()

    dst, _ := os.Create("dest.txt")
    defer dst.Close()

    // io.Copy reads from src and writes to dst in chunks — memory efficient
    bytesWritten, err := io.Copy(dst, src)
    if err != nil {
        fmt.Println("Error:", err)
        return
    }
    fmt.Printf("Copied %d bytes\n", bytesWritten)
}
```

### strings.NewReader — Mengubah String Menjadi Reader

```go
import (
    "io"
    "os"
    "strings"
)

func main() {
    r := strings.NewReader("Hello from a string reader!\n")

    // Now r implements io.Reader — use it anywhere a reader is expected
    io.Copy(os.Stdout, r)
}
```

### bytes.Buffer — Buffer Baca/Tulis di Memori

```go
import (
    "bytes"
    "fmt"
)

func main() {
    var buf bytes.Buffer

    // Write to buffer
    buf.WriteString("Hello, ")
    buf.WriteString("World!")
    fmt.Fprintf(&buf, " Count: %d", 42) // Fprintf writes to any Writer

    // Read from buffer
    fmt.Println(buf.String())   // Hello, World! Count: 42
    fmt.Println(buf.Len())      // byte length
}
```

## 1.4 Package `bufio` — Buffered I/O

Pemanggilan `Read`/`Write` langsung ke file membuat satu syscall per pemanggilan — mahal. `bufio` membungkus reader/writer dengan buffer untuk mengelompokkan operasi.

### Membaca Baris per Baris

```go
import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    f, err := os.Open("data.txt")
    if err != nil {
        fmt.Println("Error:", err)
        return
    }
    defer f.Close()

    scanner := bufio.NewScanner(f)

    lineNum := 0
    for scanner.Scan() {
        lineNum++
        line := scanner.Text()
        fmt.Printf("%d: %s\n", lineNum, line)
    }

    if err := scanner.Err(); err != nil {
        fmt.Println("Scanner error:", err)
    }
}
```

### Membaca dari Stdin

```go
func main() {
    scanner := bufio.NewScanner(os.Stdin)
    fmt.Print("Enter your name: ")

    if scanner.Scan() {
        name := scanner.Text()
        fmt.Println("Hello,", name)
    }
}
```

### Penulisan dengan Buffer

```go
import (
    "bufio"
    "os"
)

func main() {
    f, _ := os.Create("output.txt")
    defer f.Close()

    // Wraps f — batches small writes into larger ones
    w := bufio.NewWriter(f)

    for i := 0; i < 1000; i++ {
        fmt.Fprintf(w, "Line %d\n", i)
    }

    // IMPORTANT: Flush writes any buffered data to the underlying writer
    w.Flush()
}
```

## 1.5 Bekerja dengan Path — `path/filepath`

Package `path/filepath` menyediakan manipulasi path file yang sadar OS. Package ini otomatis menggunakan separator yang benar (`/` di Unix, `\` di Windows). **Jangan pernah membangun path file dengan penggabungan string** — gunakan `filepath.Join` sebagai gantinya.

Fungsi-fungsi utama:
- `filepath.Join` — menggabungkan komponen path dengan aman
- `filepath.WalkDir` — menelusuri pohon direktori (lebih dianjurkan daripada `filepath.Walk` yang lebih lama sejak Go 1.16, karena menghindari pemanggilan `os.Stat` pada setiap file)
- `filepath.Glob` — mencari file yang cocok dengan pola (misal, `*.go`)
- `filepath.Abs`, `filepath.Rel` — konversi antara path absolut dan relatif

```go
import (
    "fmt"
    "path/filepath"
)

func main() {
    // Join path components (handles OS separators)
    p := filepath.Join("home", "user", "docs", "file.txt")
    fmt.Println(p) // home/user/docs/file.txt (or home\user\... on Windows)

    // Split into directory and filename
    dir, file := filepath.Split("/home/user/docs/file.txt")
    fmt.Println("Dir:", dir)   // /home/user/docs/
    fmt.Println("File:", file) // file.txt

    // Get just the extension
    ext := filepath.Ext("photo.jpg")
    fmt.Println("Ext:", ext) // .jpg

    // Get the base name
    base := filepath.Base("/home/user/docs/file.txt")
    fmt.Println("Base:", base) // file.txt

    // Absolute path
    abs, _ := filepath.Abs("./relative/path")
    fmt.Println("Abs:", abs)

    // Walk a directory tree
    filepath.Walk(".", func(path string, info os.FileInfo, err error) error {
        if err != nil {
            return err
        }
        if !info.IsDir() {
            fmt.Println(path)
        }
        return nil
    })
}
```

# Minggu 2 — JSON, HTTP Client & API Eksternal

## 2.1 Package `encoding/json`

JSON adalah bahasa universal API web. Package `encoding/json` milik Go menangani encoding (Go ke JSON) dan decoding (JSON ke Go).

### Encoding (Marshal)

```go
import (
    "encoding/json"
    "fmt"
)

type User struct {
    ID       int    `json:"id"`
    Name     string `json:"name"`
    Email    string `json:"email"`
    Password string `json:"-"`          // excluded from JSON
    Age      int    `json:"age,omitempty"` // omitted if zero value
}

func main() {
    user := User{
        ID:       1,
        Name:     "Alice",
        Email:    "alice@example.com",
        Password: "secret",
        Age:      0, // omitempty — will not appear in JSON
    }

    // Marshal to []byte
    data, err := json.Marshal(user)
    if err != nil {
        fmt.Println("Error:", err)
        return
    }
    fmt.Println(string(data))
    // {"id":1,"name":"Alice","email":"alice@example.com"}

    // Marshal with indentation (for human readability)
    pretty, _ := json.MarshalIndent(user, "", "  ")
    fmt.Println(string(pretty))
}
```

**Penjelasan struct tag:**
- `json:"name"` — gunakan "name" sebagai key JSON
- `json:"-"` — selalu kecualikan field ini
- `json:"name,omitempty"` — kecualikan jika field bernilai zero value
- `json:"name,string"` — encode sebagai JSON string meskipun nilainya angka

### Decoding (Unmarshal)

```go
func main() {
    data := []byte(`{
        "id": 42,
        "name": "Bob",
        "email": "bob@example.com"
    }`)

    var user User
    err := json.Unmarshal(data, &user)
    if err != nil {
        fmt.Println("Error:", err)
        return
    }
    fmt.Printf("ID: %d, Name: %s\n", user.ID, user.Name)
}
```

### Encoding/Decoding dengan Stream

Untuk HTTP body dan file, gunakan streaming encoder/decoder — ini tidak memuat semua data ke memori sekaligus:

```go
import (
    "encoding/json"
    "os"
)

// Write JSON to a file
func writeJSON(filename string, v any) error {
    f, err := os.Create(filename)
    if err != nil {
        return err
    }
    defer f.Close()

    encoder := json.NewEncoder(f)
    encoder.SetIndent("", "  ")  // pretty print
    return encoder.Encode(v)
}

// Read JSON from a file
func readJSON(filename string, v any) error {
    f, err := os.Open(filename)
    if err != nil {
        return err
    }
    defer f.Close()

    return json.NewDecoder(f).Decode(v)
}
```

### Bekerja dengan JSON yang Tidak Diketahui Strukturnya — `map` dan `json.RawMessage`

```go
// When you don't know the structure, decode into a map
func main() {
    data := []byte(`{"name":"Alice","scores":[95,87,92],"active":true}`)

    var result map[string]any
    json.Unmarshal(data, &result)

    fmt.Println(result["name"])   // Alice
    fmt.Println(result["active"]) // true

    // Type assertion needed for nested values
    scores := result["scores"].([]any)
    for _, s := range scores {
        fmt.Println(s.(float64)) // JSON numbers are float64
    }
}

// json.RawMessage — delay decoding part of a JSON payload
type APIResponse struct {
    Status string          `json:"status"`
    Data   json.RawMessage `json:"data"` // keep as raw JSON bytes
}

func decode(body []byte, target any) error {
    var resp APIResponse
    if err := json.Unmarshal(body, &resp); err != nil {
        return err
    }
    if resp.Status != "ok" {
        return fmt.Errorf("API error: %s", resp.Status)
    }
    return json.Unmarshal(resp.Data, target)
}
```

### Custom JSON Marshaling

Implementasikan `json.Marshaler` dan `json.Unmarshaler` untuk kontrol penuh:

```go
import "time"

type Date struct {
    time.Time
}

// Custom format when marshaling
func (d Date) MarshalJSON() ([]byte, error) {
    return json.Marshal(d.Format("2006-01-02"))
}

// Custom parsing when unmarshaling
func (d *Date) UnmarshalJSON(data []byte) error {
    var s string
    if err := json.Unmarshal(data, &s); err != nil {
        return err
    }
    t, err := time.Parse("2006-01-02", s)
    if err != nil {
        return err
    }
    d.Time = t
    return nil
}

type Event struct {
    Name string `json:"name"`
    Date Date   `json:"date"`
}

// {"name":"Launch","date":"2024-03-15"}
```

## 2.2 HTTP Client `net/http`

Package `net/http` milik Go mencakup server dan client. Mari mulai dengan client.

### Request GET Sederhana

```go
import (
    "fmt"
    "io"
    "net/http"
)

func main() {
    resp, err := http.Get("https://api.github.com/users/golang")
    if err != nil {
        fmt.Println("Error:", err)
        return
    }
    defer resp.Body.Close() // ALWAYS close the body

    fmt.Println("Status:", resp.Status)         // 200 OK
    fmt.Println("Content-Type:", resp.Header.Get("Content-Type"))

    body, err := io.ReadAll(resp.Body)
    if err != nil {
        fmt.Println("Error reading body:", err)
        return
    }
    fmt.Println(string(body))
}
```

### Mendekode Response JSON

```go
type GithubUser struct {
    Login     string `json:"login"`
    Name      string `json:"name"`
    Followers int    `json:"followers"`
    PublicRepos int  `json:"public_repos"`
}

func getGithubUser(username string) (*GithubUser, error) {
    url := fmt.Sprintf("https://api.github.com/users/%s", username)

    resp, err := http.Get(url)
    if err != nil {
        return nil, fmt.Errorf("request failed: %w", err)
    }
    defer resp.Body.Close()

    if resp.StatusCode != http.StatusOK {
        return nil, fmt.Errorf("unexpected status: %s", resp.Status)
    }

    var user GithubUser
    if err := json.NewDecoder(resp.Body).Decode(&user); err != nil {
        return nil, fmt.Errorf("decode failed: %w", err)
    }

    return &user, nil
}
```

### Menggunakan HTTP Client Kustom

Shortcut `http.Get` menggunakan client default tanpa timeout. Di production, **selalu** atur timeout:

```go
client := &http.Client{
    Timeout: 10 * time.Second,
}

resp, err := client.Get("https://api.example.com/data")
```

### Request POST dengan Body JSON

```go
type CreateUserRequest struct {
    Name  string `json:"name"`
    Email string `json:"email"`
}

func createUser(client *http.Client, user CreateUserRequest) error {
    body, err := json.Marshal(user)
    if err != nil {
        return err
    }

    req, err := http.NewRequest("POST", "https://api.example.com/users",
        bytes.NewReader(body))
    if err != nil {
        return err
    }

    req.Header.Set("Content-Type", "application/json")
    req.Header.Set("Authorization", "Bearer "+os.Getenv("API_TOKEN"))

    resp, err := client.Do(req)
    if err != nil {
        return fmt.Errorf("request failed: %w", err)
    }
    defer resp.Body.Close()

    if resp.StatusCode != http.StatusCreated {
        body, _ := io.ReadAll(resp.Body)
        return fmt.Errorf("API error %d: %s", resp.StatusCode, string(body))
    }

    return nil
}
```

### Membuat Wrapper HTTP Client

Membungkus client menghindari pengulangan header dan base URL:

```go
type APIClient struct {
    baseURL    string
    httpClient *http.Client
    token      string
}

func NewAPIClient(baseURL, token string) *APIClient {
    return &APIClient{
        baseURL: baseURL,
        token:   token,
        httpClient: &http.Client{
            Timeout: 15 * time.Second,
        },
    }
}

func (c *APIClient) do(method, path string, body io.Reader) (*http.Response, error) {
    req, err := http.NewRequest(method, c.baseURL+path, body)
    if err != nil {
        return nil, err
    }
    req.Header.Set("Authorization", "Bearer "+c.token)
    req.Header.Set("Content-Type", "application/json")
    return c.httpClient.Do(req)
}

func (c *APIClient) Get(path string, out any) error {
    resp, err := c.do("GET", path, nil)
    if err != nil {
        return err
    }
    defer resp.Body.Close()
    return json.NewDecoder(resp.Body).Decode(out)
}
```

## 2.3 Membangun URL & Query Parameter

Selalu gunakan package `net/url` untuk membangun URL alih-alih `fmt.Sprintf` atau penggabungan string. Pembuatan string secara naif akan rusak ketika nilai mengandung karakter khusus seperti `&`, `=`, atau spasi — `net/url` menangani encoding yang tepat secara otomatis.

Juga berguna: `url.PathEscape` dan `url.QueryEscape` untuk meng-encode segmen path atau nilai query individual ketika kamu butuh kontrol manual.

```go
import "net/url"

func buildURL(base, path string, params map[string]string) (string, error) {
    u, err := url.Parse(base)
    if err != nil {
        return "", err
    }

    u.Path = path

    q := u.Query() // returns url.Values (map[string][]string)
    for k, v := range params {
        q.Set(k, v)
    }
    u.RawQuery = q.Encode()

    return u.String(), nil
}

// Usage
urlStr, _ := buildURL(
    "https://api.example.com",
    "/search",
    map[string]string{
        "q":     "golang",
        "limit": "10",
        "page":  "2",
    },
)
fmt.Println(urlStr)
// https://api.example.com/search?limit=10&page=2&q=golang
```

# Minggu 3 — Membangun HTTP Server

## 3.1 Server `net/http`

Standard library Go sudah termasuk HTTP server yang siap production. Tidak perlu framework untuk belajar.

### Server Hello World

```go
package main

import (
    "fmt"
    "net/http"
)

func helloHandler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintln(w, "Hello, World!")
}

func main() {
    http.HandleFunc("/", helloHandler)

    fmt.Println("Server listening on :8080")
    if err := http.ListenAndServe(":8080", nil); err != nil {
        fmt.Println("Server error:", err)
    }
}
```

Signature handler selalu:
```go
func(w http.ResponseWriter, r *http.Request)
```

- `w http.ResponseWriter` — tulis response kamu di sini (status, header, body)
- `r *http.Request` — request yang masuk (URL, method, header, body)

### Objek Request

```go
func handler(w http.ResponseWriter, r *http.Request) {
    // Method: GET, POST, PUT, DELETE, etc.
    fmt.Println("Method:", r.Method)

    // URL path and query
    fmt.Println("Path:", r.URL.Path)
    fmt.Println("Query:", r.URL.RawQuery)

    // Query parameters
    q := r.URL.Query()
    name := q.Get("name")      // single value
    tags := q["tag"]           // multiple values for same key

    // Headers
    contentType := r.Header.Get("Content-Type")
    auth := r.Header.Get("Authorization")

    // Remote address
    fmt.Println("Client IP:", r.RemoteAddr)
}
```

### Menulis Response

```go
func handler(w http.ResponseWriter, r *http.Request) {
    // Set headers BEFORE writing status or body
    w.Header().Set("Content-Type", "application/json")
    w.Header().Set("X-Request-ID", "abc123")

    // Write status code — once you write body, status is locked in
    w.WriteHeader(http.StatusCreated) // 201

    // Write body
    w.Write([]byte(`{"message":"created"}`))
}

// Helper: write JSON response
func writeJSON(w http.ResponseWriter, status int, v any) {
    w.Header().Set("Content-Type", "application/json")
    w.WriteHeader(status)
    json.NewEncoder(w).Encode(v)
}

// Helper: write JSON error
func writeError(w http.ResponseWriter, status int, message string) {
    writeJSON(w, status, map[string]string{"error": message})
}
```

### ServeMux — Routing Request

`http.ServeMux` mengarahkan request yang masuk ke handler berdasarkan URL path.

```go
func main() {
    mux := http.NewServeMux()

    // Exact path match
    mux.HandleFunc("/", homeHandler)

    // Prefix match (trailing slash matches all sub-paths)
    mux.HandleFunc("/api/", apiHandler)

    // Static files
    fs := http.FileServer(http.Dir("./static"))
    mux.Handle("/static/", http.StripPrefix("/static/", fs))

    server := &http.Server{
        Addr:         ":8080",
        Handler:      mux,
        ReadTimeout:  10 * time.Second,
        WriteTimeout: 10 * time.Second,
        IdleTimeout:  60 * time.Second,
    }

    fmt.Println("Listening on :8080")
    if err := server.ListenAndServe(); err != nil {
        fmt.Println("Error:", err)
    }
}
```

**Catatan tentang routing Go 1.22:** Go 1.22 menambahkan pattern matching ke `ServeMux`. Jika versi Go kamu 1.22+, kamu bisa menggunakan:

```go
mux.HandleFunc("GET /users/{id}", getUserHandler)
mux.HandleFunc("POST /users", createUserHandler)
mux.HandleFunc("DELETE /users/{id}", deleteUserHandler)

// Extract path parameters
func getUserHandler(w http.ResponseWriter, r *http.Request) {
    id := r.PathValue("id")
    fmt.Println("User ID:", id)
}
```

## 3.2 Membangun REST API

Mari bangun REST API lengkap untuk mengelola daftar buku. Kita akan menggunakan penyimpanan in-memory untuk saat ini (database akan dibahas di Minggu 4).

### Model Data & Penyimpanan

```go
// models.go
package main

import (
    "sync"
    "time"
)

type Book struct {
    ID        int       `json:"id"`
    Title     string    `json:"title"`
    Author    string    `json:"author"`
    Year      int       `json:"year"`
    CreatedAt time.Time `json:"created_at"`
}

// In-memory store — thread-safe
type BookStore struct {
    mu     sync.RWMutex
    books  map[int]*Book
    nextID int
}

func NewBookStore() *BookStore {
    return &BookStore{
        books:  make(map[int]*Book),
        nextID: 1,
    }
}

func (s *BookStore) Create(b *Book) *Book {
    s.mu.Lock()
    defer s.mu.Unlock()
    b.ID = s.nextID
    b.CreatedAt = time.Now()
    s.books[b.ID] = b
    s.nextID++
    return b
}

func (s *BookStore) GetByID(id int) (*Book, bool) {
    s.mu.RLock()
    defer s.mu.RUnlock()
    b, ok := s.books[id]
    return b, ok
}

func (s *BookStore) List() []*Book {
    s.mu.RLock()
    defer s.mu.RUnlock()
    result := make([]*Book, 0, len(s.books))
    for _, b := range s.books {
        result = append(result, b)
    }
    return result
}

func (s *BookStore) Update(id int, updated *Book) (*Book, bool) {
    s.mu.Lock()
    defer s.mu.Unlock()
    b, ok := s.books[id]
    if !ok {
        return nil, false
    }
    b.Title = updated.Title
    b.Author = updated.Author
    b.Year = updated.Year
    return b, true
}

func (s *BookStore) Delete(id int) bool {
    s.mu.Lock()
    defer s.mu.Unlock()
    if _, ok := s.books[id]; !ok {
        return false
    }
    delete(s.books, id)
    return true
}
```

### Handler

```go
// handlers.go
package main

import (
    "encoding/json"
    "net/http"
    "strconv"
    "strings"
)

type BookHandler struct {
    store *BookStore
}

func NewBookHandler(store *BookStore) *BookHandler {
    return &BookHandler{store: store}
}

func (h *BookHandler) ServeHTTP(w http.ResponseWriter, r *http.Request) {
    // Route: /books or /books/{id}
    path := strings.TrimPrefix(r.URL.Path, "/books")
    path = strings.TrimPrefix(path, "/")

    if path == "" {
        // /books
        switch r.Method {
        case http.MethodGet:
            h.listBooks(w, r)
        case http.MethodPost:
            h.createBook(w, r)
        default:
            writeError(w, http.StatusMethodNotAllowed, "method not allowed")
        }
        return
    }

    // /books/{id}
    id, err := strconv.Atoi(path)
    if err != nil {
        writeError(w, http.StatusBadRequest, "invalid book ID")
        return
    }

    switch r.Method {
    case http.MethodGet:
        h.getBook(w, r, id)
    case http.MethodPut:
        h.updateBook(w, r, id)
    case http.MethodDelete:
        h.deleteBook(w, r, id)
    default:
        writeError(w, http.StatusMethodNotAllowed, "method not allowed")
    }
}

func (h *BookHandler) listBooks(w http.ResponseWriter, r *http.Request) {
    books := h.store.List()
    writeJSON(w, http.StatusOK, books)
}

func (h *BookHandler) createBook(w http.ResponseWriter, r *http.Request) {
    var book Book
    if err := json.NewDecoder(r.Body).Decode(&book); err != nil {
        writeError(w, http.StatusBadRequest, "invalid request body")
        return
    }

    if book.Title == "" || book.Author == "" {
        writeError(w, http.StatusBadRequest, "title and author are required")
        return
    }

    created := h.store.Create(&book)
    writeJSON(w, http.StatusCreated, created)
}

func (h *BookHandler) getBook(w http.ResponseWriter, r *http.Request, id int) {
    book, ok := h.store.GetByID(id)
    if !ok {
        writeError(w, http.StatusNotFound, "book not found")
        return
    }
    writeJSON(w, http.StatusOK, book)
}

func (h *BookHandler) updateBook(w http.ResponseWriter, r *http.Request, id int) {
    var updated Book
    if err := json.NewDecoder(r.Body).Decode(&updated); err != nil {
        writeError(w, http.StatusBadRequest, "invalid request body")
        return
    }

    book, ok := h.store.Update(id, &updated)
    if !ok {
        writeError(w, http.StatusNotFound, "book not found")
        return
    }
    writeJSON(w, http.StatusOK, book)
}

func (h *BookHandler) deleteBook(w http.ResponseWriter, r *http.Request, id int) {
    if !h.store.Delete(id) {
        writeError(w, http.StatusNotFound, "book not found")
        return
    }
    w.WriteHeader(http.StatusNoContent) // 204 — success, no body
}
```

## 3.3 Middleware

Middleware membungkus handler untuk menambahkan perilaku — logging, auth, rate limiting, CORS — tanpa mengotori logika handler kamu.

### Pola Middleware

```go
// A middleware takes a handler and returns a handler
type Middleware func(http.Handler) http.Handler

// Logging middleware
func Logger(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        start := time.Now()
        fmt.Printf("%s %s ", r.Method, r.URL.Path)

        next.ServeHTTP(w, r) // call the next handler

        fmt.Printf("%v\n", time.Since(start))
    })
}

// Recovery middleware — catches panics so the server doesn't crash
func Recovery(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        defer func() {
            if err := recover(); err != nil {
                fmt.Printf("panic: %v\n", err)
                writeError(w, http.StatusInternalServerError, "internal server error")
            }
        }()
        next.ServeHTTP(w, r)
    })
}

// Auth middleware — checks for a token
func RequireAuth(token string) Middleware {
    return func(next http.Handler) http.Handler {
        return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
            auth := r.Header.Get("Authorization")
            if auth != "Bearer "+token {
                writeError(w, http.StatusUnauthorized, "unauthorized")
                return
            }
            next.ServeHTTP(w, r)
        })
    }
}
```

### Merangkai Middleware

```go
// Chain applies middleware in order (outermost first)
func Chain(h http.Handler, middlewares ...Middleware) http.Handler {
    for i := len(middlewares) - 1; i >= 0; i-- {
        h = middlewares[i](h)
    }
    return h
}

func main() {
    store := NewBookStore()
    bookHandler := NewBookHandler(store)

    mux := http.NewServeMux()
    mux.Handle("/books", bookHandler)
    mux.Handle("/books/", bookHandler)

    // Wrap the entire mux with middleware
    handler := Chain(mux,
        Recovery,
        Logger,
        RequireAuth(os.Getenv("API_TOKEN")),
    )

    http.ListenAndServe(":8080", handler)
}
```

### Menangkap Status Response di Middleware

`http.ResponseWriter` standar tidak mengekspos status code setelah ditulis. Gunakan wrapper:

```go
type responseWriter struct {
    http.ResponseWriter
    status int
    size   int
}

func (rw *responseWriter) WriteHeader(status int) {
    rw.status = status
    rw.ResponseWriter.WriteHeader(status)
}

func (rw *responseWriter) Write(b []byte) (int, error) {
    if rw.status == 0 {
        rw.status = http.StatusOK
    }
    n, err := rw.ResponseWriter.Write(b)
    rw.size += n
    return n, err
}

func Logger(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        rw := &responseWriter{ResponseWriter: w}
        start := time.Now()
        next.ServeHTTP(rw, r)
        fmt.Printf("%s %s → %d (%v)\n",
            r.Method, r.URL.Path, rw.status, time.Since(start))
    })
}
```

Ambil server buku dari minggu ini dan tambahkan:

## 4.3 Operasi CRUD

### Query Banyak Baris

```go
func (s *Store) ListBooks() ([]Book, error) {
    rows, err := s.db.Query(
        "SELECT id, title, author, year, created_at FROM books ORDER BY id",
    )
    if err != nil {
        return nil, fmt.Errorf("listBooks: %w", err)
    }
    defer rows.Close() // always close rows

    var books []Book
    for rows.Next() {
        var b Book
        err := rows.Scan(&b.ID, &b.Title, &b.Author, &b.Year, &b.CreatedAt)
        if err != nil {
            return nil, fmt.Errorf("listBooks scan: %w", err)
        }
        books = append(books, b)
    }

    // Check for errors that occurred during iteration
    if err := rows.Err(); err != nil {
        return nil, fmt.Errorf("listBooks rows: %w", err)
    }

    return books, nil
}
```

### Query Satu Baris

```go
func (s *Store) GetBook(id int) (*Book, error) {
    var b Book
    err := s.db.QueryRow(
        "SELECT id, title, author, year, created_at FROM books WHERE id = $1",
        id,
    ).Scan(&b.ID, &b.Title, &b.Author, &b.Year, &b.CreatedAt)

    if err == sql.ErrNoRows {
        return nil, fmt.Errorf("book %d: %w", id, ErrNotFound)
    }
    if err != nil {
        return nil, fmt.Errorf("getBook: %w", err)
    }

    return &b, nil
}

var ErrNotFound = errors.New("not found")
```

### Memasukkan Data

```go
func (s *Store) CreateBook(b *Book) (*Book, error) {
    err := s.db.QueryRow(

         VALUES ($1, $2, $3)
```

### Memperbarui Data

```go
func (s *Store) UpdateBook(id int, b *Book) (*Book, error) {
    result, err := s.db.Exec(
        `UPDATE books SET title = $1, author = $2, year = $3
         WHERE id = $4`,
        b.Title, b.Author, b.Year, id,
    )
    if err != nil {
        return nil, fmt.Errorf("updateBook: %w", err)
    }

    rows, err := result.RowsAffected()
    if err != nil {
        return nil, err
    }
    if rows == 0 {
        return nil, fmt.Errorf("book %d: %w", id, ErrNotFound)
    }

    b.ID = id
    return b, nil
}
```

### Menghapus Data

```go
func (s *Store) DeleteBook(id int) error {
    result, err := s.db.Exec("DELETE FROM books WHERE id = $1", id)
    if err != nil {
        return fmt.Errorf("deleteBook: %w", err)
    }

    rows, _ := result.RowsAffected()
    if rows == 0 {
        return fmt.Errorf("book %d: %w", id, ErrNotFound)
    }
    return nil
}
```

## 4.4 Transaksi

Transaksi mengelompokkan beberapa operasi — semua berhasil atau semua gagal.

```go
func (s *Store) TransferStock(fromID, toID, quantity int) error {
    // Begin a transaction
    tx, err := s.db.Begin()
    if err != nil {
        return fmt.Errorf("begin transaction: %w", err)
    }

    // Defer a rollback — if we return early due to error, this runs.
    // If we commit successfully, Rollback is a no-op.
    defer tx.Rollback()

    // Deduct from source
    _, err = tx.Exec(
        "UPDATE inventory SET quantity = quantity - $1 WHERE id = $2 AND quantity >= $1",
        quantity, fromID,
    )
    if err != nil {
        return fmt.Errorf("deduct stock: %w", err)
    }

    // Add to destination
    _, err = tx.Exec(
        "UPDATE inventory SET quantity = quantity + $1 WHERE id = $2",
        quantity, toID,
    )
    if err != nil {
        return fmt.Errorf("add stock: %w", err)
    }

    // Commit — this is the only success path
    if err := tx.Commit(); err != nil {
        return fmt.Errorf("commit: %w", err)
    }

    return nil
}
```

**Praktik terbaik untuk transaksi:**

```go
// Prefer the context-aware version in production:
tx, err := s.db.BeginTx(ctx, nil)  // nil = default isolation level

// Always check RowsAffected for conditional updates:
result, err := tx.Exec(
    "UPDATE inventory SET quantity = quantity - $1 WHERE id = $2 AND quantity >= $1",
    quantity, fromID,
)
if err != nil {
    return err
}
rows, _ := result.RowsAffected()
if rows == 0 {
    return fmt.Errorf("insufficient stock for item %d", fromID)
}
```

> **Isolation level:** Secara default, PostgreSQL menggunakan `READ COMMITTED`. Kamu bisa mengoper `&sql.TxOptions{Isolation: sql.LevelSerializable}` ke `BeginTx` untuk isolasi yang lebih ketat saat diperlukan (misal, transaksi keuangan). Isolasi yang lebih tinggi mencegah lebih banyak anomali konkurensi tapi bisa mengurangi throughput.

## 4.5 Menggunakan `sqlx` — Wrapper Tipis

`sqlx` memperluas `database/sql` dengan method-method yang lebih praktis. Install:

```go
import "github.com/jmoiron/sqlx"

db, err := sqlx.Connect("postgres", dsn)

// Get — scan a single row into a struct automatically
var book Book
err = db.Get(&book, "SELECT * FROM books WHERE id = $1", 1)

// Select — scan multiple rows into a slice automatically
var books []Book
err = db.Select(&books, "SELECT * FROM books ORDER BY id")

// NamedExec — use struct field names as placeholders
book := Book{Title: "Go", Author: "Donovan", Year: 2015}
_, err = db.NamedExec(
    "INSERT INTO books (title, author, year) VALUES (:title, :author, :year)",

)
```

```go
type Book struct {
```

## 4.6 Migrasi Database

Migrasi melacak perubahan skema dalam file yang dikontrol versi. Gunakan `golang-migrate`:

```bash
go install -tags 'postgres' github.com/golang-migrate/migrate/v4/cmd/migrate@latest
```

Buat file migrasi:

```bash
migrate create -ext sql -dir migrations -seq create_books_table
# Creates:
# migrations/000001_create_books_table.up.sql
# migrations/000001_create_books_table.down.sql
```

```sql
-- migrations/000001_create_books_table.up.sql
CREATE TABLE books (
    id         SERIAL PRIMARY KEY,
    title      TEXT NOT NULL,
    author     TEXT NOT NULL,
    year       INTEGER,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- migrations/000001_create_books_table.down.sql
DROP TABLE IF EXISTS books;
```

Jalankan migrasi:

```bash
migrate -path migrations -database "$DATABASE_URL" up
migrate -path migrations -database "$DATABASE_URL" down 1
```

Jalankan migrasi dari kode Go:

```go
import "github.com/golang-migrate/migrate/v4"

func runMigrations(dsn string) error {
    m, err := migrate.New("file://migrations", dsn)
    if err != nil {
        return err
    }
    if err := m.Up(); err != nil && err != migrate.ErrNoChange {
        return err
    }
    return nil
}
```

## Minggu 4 — Proyek

**Hubungkan API Buku ke PostgreSQL**

Ambil API Minggu 3 dan ganti penyimpanan in-memory dengan database PostgreSQL sungguhan.

Persyaratan:
1. Buat struct `Store` yang membungkus `*sql.DB` (atau `*sqlx.DB`)
2. Implementasikan semua operasi CRUD menggunakan SQL
3. Gunakan placeholder `$1`, `$2` (bukan string formatting — risiko SQL injection)
4. Tangani `sql.ErrNoRows` dan kembalikan response 404 yang tepat

# Minggu 5 — Context, Middleware & Tools CLI

## 5.1 Package `context`

`context` membawa deadline, sinyal pembatalan, dan nilai request-scoped melewati batas API. Ini adalah cara standar untuk menangani timeout dan pembatalan di Go.

### Dasar-Dasar Context

```go
import "context"

// Background — the root context, never cancelled
ctx := context.Background()

// TODO — placeholder when you're not sure which context to use yet
ctx := context.TODO()

// With timeout — auto-cancels after duration
ctx, cancel := context.WithTimeout(ctx, 5*time.Second)
defer cancel() // always call cancel to free resources

// With deadline — cancels at a specific time
deadline := time.Now().Add(10 * time.Second)
ctx, cancel := context.WithDeadline(ctx, deadline)
defer cancel()

// With cancel — manual cancellation
ctx, cancel := context.WithCancel(ctx)
defer cancel()
go func() {
    time.Sleep(3 * time.Second)
    cancel() // cancel from anywhere
}()

// Check if context is done
select {
case <-ctx.Done():
    fmt.Println("context cancelled:", ctx.Err())
default:
    fmt.Println("still running")
}
```

### Context dengan HTTP Request

```go
func fetchData(ctx context.Context, url string) ([]byte, error) {
    req, err := http.NewRequestWithContext(ctx, "GET", url, nil)
    if err != nil {
        return nil, err
    }

    resp, err := http.DefaultClient.Do(req)
    if err != nil {
        // If context was cancelled, err wraps context.DeadlineExceeded
        return nil, fmt.Errorf("fetch failed: %w", err)
    }
    defer resp.Body.Close()

    return io.ReadAll(resp.Body)
}

func main() {
    ctx, cancel := context.WithTimeout(context.Background(), 3*time.Second)
    defer cancel()

    data, err := fetchData(ctx, "https://api.example.com/data")
    if err != nil {
        if errors.Is(err, context.DeadlineExceeded) {
            fmt.Println("Request timed out")
        } else {
            fmt.Println("Error:", err)
        }
        return
    }
    fmt.Println(string(data))
}
```

### Context di HTTP Handler

Setiap `http.Request` membawa context. Gunakan untuk meneruskan pembatalan ke query database dan panggilan downstream kamu:

```go
func (h *BookHandler) getBook(w http.ResponseWriter, r *http.Request) {
    ctx := r.Context() // get context from request

    id, _ := strconv.Atoi(r.PathValue("id"))

    // Pass context to store — if client disconnects, query is cancelled
    book, err := h.store.GetBook(ctx, id)
    if errors.Is(err, context.Canceled) {
        return // client disconnected, no need to respond
    }
    if errors.Is(err, ErrNotFound) {
        writeError(w, http.StatusNotFound, "book not found")
        return
    }
    if err != nil {
        writeError(w, http.StatusInternalServerError, "database error")
        return
    }

    writeJSON(w, http.StatusOK, book)
}

// Store method accepts context
func (s *Store) GetBook(ctx context.Context, id int) (*Book, error) {
    var b Book
    err := s.db.QueryRowContext(ctx,
        "SELECT id, title, author, year FROM books WHERE id = $1", id,
    ).Scan(&b.ID, &b.Title, &b.Author, &b.Year)

    if err == sql.ErrNoRows {
        return nil, ErrNotFound
    }
    return &b, err
}
```

**Aturan:** Selalu oper `context.Context` sebagai parameter pertama dari fungsi apa pun yang melakukan I/O (database, HTTP, file). Jangan pernah menyimpan context di dalam struct.

### Context Value

Simpan data request-scoped (user ID dari auth, request ID untuk logging):

## 5.2 Structured Logging dengan `log/slog`

Go 1.21 memperkenalkan `log/slog` — structured logging dengan level di standard library.

**Level log dan kapan menggunakannya:**
- `slog.Debug` — detail state internal, hanya aktifkan saat development/debugging
- `slog.Info` — operasi normal: server dimulai, request diproses, user dibuat
- `slog.Warn` — sesuatu yang tidak diharapkan tapi sistem masih berjalan: memori tinggi, query lambat, perlu retry
- `slog.Error` — sesuatu gagal: error database, kegagalan API eksternal, state yang tidak bisa dipulihkan

```go
import "log/slog"

func main() {
    // Default logger — outputs to stderr
    slog.Info("server started", "port", 8080)
    slog.Warn("high memory", "usage_pct", 85)
    slog.Error("database error", "err", err, "query", "SELECT...")

    // JSON logger — great for production/log aggregation
    logger := slog.New(slog.NewJSONHandler(os.Stdout, &slog.HandlerOptions{
        Level: slog.LevelInfo,
    }))
    slog.SetDefault(logger)

    // Now slog.Info, slog.Error etc. use this logger
    slog.Info("request received",
        "method", "GET",
        "path", "/books",
        "status", 200,
        "latency_ms", 12,
    )
    // {"time":"2024-01-15T10:23:01Z","level":"INFO","msg":"request received","method":"GET","path":"/books","status":200,"latency_ms":12}

    // With persistent context — slog.With creates a child logger
    reqLogger := logger.With("request_id", "abc-123")
    reqLogger.Info("processing request")
    reqLogger.Info("query complete", "rows", 42)
    // Both log lines include request_id — useful for tracing a request through multiple operations

    // For development, use TextHandler for human-readable output:
    devLogger := slog.New(slog.NewTextHandler(os.Stdout, &slog.HandlerOptions{
        Level: slog.LevelDebug,
    }))
    devLogger.Debug("this only appears with LevelDebug")
    // time=2024-01-15T10:23:01Z level=DEBUG msg="this only appears with LevelDebug"
}
```

```go
        JWTSecret:    getEnv("JWT_SECRET", ""),

        WriteTimeout: getDurationEnv("WRITE_TIMEOUT", 10*time.Second),

func getEnv(key, fallback string) string {

        return v

    return fallback

    if v == "" {
        return fallback

        return fallback
    }
    return d
```

## 5.4 Membangun Tools CLI dengan `flag`

Package `flag` mem-parse flag command-line. Tidak seperti parsing `os.Args` mentah (yang mengharuskan kamu memeriksa posisi secara manual dan menangani argumen yang hilang), `flag` memberikan parsing bertipe, nilai default, dan output `--help` otomatis.

**Keterbatasan `flag`:** Tidak mendukung flag pendek (`-v` sebagai singkatan `--verbose`) atau sintaks POSIX-style `--flag=value`.
```go
import "flag"

func main() {
    // Define flags
    port := flag.Int("port", 8080, "server port")
    host := flag.String("host", "localhost", "server host")
    verbose := flag.Bool("verbose", false, "enable verbose output")
    timeout := flag.Duration("timeout", 30*time.Second, "request timeout")

    // Parse — must call before using any flag values
    flag.Parse()

    // Remaining non-flag arguments
    args := flag.Args()

    fmt.Printf("Starting on %s:%d (verbose=%v, timeout=%v)\n",
        *host, *port, *verbose, *timeout)
    fmt.Println("Extra args:", args)
}
```

## 5.5 Graceful Shutdown

Ketika server menerima SIGTERM (dari Docker, Kubernetes, atau Ctrl+C), server harus:
1. Berhenti menerima koneksi baru
2. Menyelesaikan request yang sedang berjalan
3. Menutup koneksi database
4. Keluar

```go
import (
    "context"
    "net/http"
    "os"
    "os/signal"
    "syscall"
    "time"
)

func main() {
    db := connectDB()
    defer db.Close()

    mux := http.NewServeMux()
    // ... register handlers

    server := &http.Server{
        Addr:    ":8080",
        Handler: mux,
    }

    // Start server in background goroutine
    go func() {
        fmt.Println("Server listening on :8080")
        if err := server.ListenAndServe(); err != http.ErrServerClosed {
            fmt.Println("Server error:", err)
            os.Exit(1)
        }
    }()

    // Wait for interrupt signal
    quit := make(chan os.Signal, 1)
    signal.Notify(quit, syscall.SIGINT, syscall.SIGTERM)
    <-quit // block until signal received

    fmt.Println("Shutting down...")

    // Give in-flight requests 30 seconds to complete
    ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
    defer cancel()

    if err := server.Shutdown(ctx); err != nil {
        fmt.Println("Forced shutdown:", err)
    }

    fmt.Println("Server stopped")
}
```

```go
package jwt

import (
    "errors"
    "time"
    "github.com/golang-jwt/jwt/v5"
)

var ErrInvalidToken = errors.New("invalid token")

type Claims struct {
    UserID int `json:"user_id"`
    jwt.RegisteredClaims
}

func Generate(userID int, secret string, expiry time.Duration) (string, error) {
    claims := Claims{
        UserID: userID,
        RegisteredClaims: jwt.RegisteredClaims{
            ExpiresAt: jwt.NewNumericDate(time.Now().Add(expiry)),
            IssuedAt:  jwt.NewNumericDate(time.Now()),
        },
    }
    token := jwt.NewWithClaims(jwt.SigningMethodHS256, claims)
    return token.SignedString([]byte(secret))
}

func Validate(tokenStr, secret string) (*Claims, error) {
    token, err := jwt.ParseWithClaims(tokenStr, &Claims{},
        func(t *jwt.Token) (any, error) {
            if _, ok := t.Method.(*jwt.SigningMethodHMAC); !ok {
                return nil, ErrInvalidToken
            }
            return []byte(secret), nil
        },
    )
    if err != nil || !token.Valid {
        return nil, ErrInvalidToken
    }
    claims, ok := token.Claims.(*Claims)
    if !ok {
        return nil, ErrInvalidToken
    }
    return claims, nil
}
```

```go
    var user model.User
    err := s.db.QueryRowContext(ctx,
        `SELECT id, email, password_hash, name, created_at FROM users WHERE email = $1`,

        return nil, ErrNotFound
```

```go
// internal/store/book.go (excerpt)
func (s *BookStore) List(ctx context.Context, f model.BookFilter) ([]model.Book, int, error) {
    // Build query dynamically
    where := []string{"1=1"}
    args := []any{}
    i := 1

    if f.Search != "" {
        where = append(where, fmt.Sprintf(
            "to_tsvector('english', title || ' ' || author) @@ plainto_tsquery($%d)", i))
        args = append(args, f.Search)
        i++
    }
    if f.Author != "" {
        where = append(where, fmt.Sprintf("author ILIKE $%d", i))
        args = append(args, "%"+f.Author+"%")
        i++
    }
    if f.MinPrice > 0 {
        where = append(where, fmt.Sprintf("price >= $%d", i))
        args = append(args, f.MinPrice)
        i++
    }

    // Count total for pagination
    countQuery := "SELECT COUNT(*) FROM books WHERE " + strings.Join(where, " AND ")
    var total int
    s.db.QueryRowContext(ctx, countQuery, args...).Scan(&total)

    // Fetch page
    limit := f.Limit
    if limit == 0 { limit = 20 }
    offset := (f.Page - 1) * limit
    if offset < 0 { offset = 0 }

    query := fmt.Sprintf(
        "SELECT id, title, author, description, price, stock, year FROM books WHERE %s ORDER BY id LIMIT $%d OFFSET $%d",
        strings.Join(where, " AND "), i, i+1,
    )
    args = append(args, limit, offset)

    rows, err := s.db.QueryContext(ctx, query, args...)
    if err != nil {
        return nil, 0, err
    }
    defer rows.Close()

    var books []model.Book
    for rows.Next() {
        var b model.Book
        rows.Scan(&b.ID, &b.Title, &b.Author, &b.Description, &b.Price, &b.Stock, &b.Year)
        books = append(books, b)
    }

    return books, total, rows.Err()
}
```

```go
type contextKey string
const UserIDKey contextKey = "user_id"

func Auth(secret string) func(http.Handler) http.Handler {
    return func(next http.Handler) http.Handler {
        return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
            header := r.Header.Get("Authorization")
            if !strings.HasPrefix(header, "Bearer ") {
                writeError(w, http.StatusUnauthorized, "missing token")
                return
            }

            token := strings.TrimPrefix(header, "Bearer ")
            claims, err := myjwt.Validate(token, secret)
            if err != nil {
                writeError(w, http.StatusUnauthorized, "invalid token")
                return
            }

            ctx := context.WithValue(r.Context(), UserIDKey, claims.UserID)
            next.ServeHTTP(w, r.WithContext(ctx))
        })
    }
}
```

### Server Utama

```go
// main.go
func main() {
    _ = godotenv.Load()

    cfg, err := config.Load()
    if err != nil {
        slog.Error("config error", "err", err)
        os.Exit(1)
    }

    logger := slog.New(slog.NewJSONHandler(os.Stdout, nil))
    slog.SetDefault(logger)

    db, err := sql.Open("postgres", cfg.DatabaseURL)
    if err != nil {
        slog.Error("db open error", "err", err)
        os.Exit(1)
    }
    defer db.Close()

    if err := db.Ping(); err != nil {
        slog.Error("db ping error", "err", err)
        os.Exit(1)
    }

    // Stores
    userStore  := store.NewUserStore(db)
    bookStore  := store.NewBookStore(db)
    orderStore := store.NewOrderStore(db)

    // Handlers
    userHandler  := handler.NewUserHandler(userStore, cfg.JWTSecret)
    bookHandler  := handler.NewBookHandler(bookStore)
    orderHandler := handler.NewOrderHandler(orderStore)

    // Router
    mux := http.NewServeMux()

    // Public routes
    mux.HandleFunc("POST /api/register", userHandler.Register)
    mux.HandleFunc("POST /api/login",    userHandler.Login)
    mux.HandleFunc("GET /api/books",     bookHandler.List)
    mux.HandleFunc("GET /api/books/{id}", bookHandler.Get)

    // Protected routes — wrap with Auth middleware
    protected := middleware.Auth(cfg.JWTSecret)
    mux.Handle("POST /api/books",          protected(http.HandlerFunc(bookHandler.Create)))
    mux.Handle("PUT /api/books/{id}",      protected(http.HandlerFunc(bookHandler.Update)))
    mux.Handle("DELETE /api/books/{id}",   protected(http.HandlerFunc(bookHandler.Delete)))
    mux.Handle("POST /api/orders",         protected(http.HandlerFunc(orderHandler.Create)))
    mux.Handle("GET /api/orders",          protected(http.HandlerFunc(orderHandler.List)))
    mux.Handle("GET /api/orders/{id}",     protected(http.HandlerFunc(orderHandler.Get)))
    mux.Handle("GET /api/profile",         protected(http.HandlerFunc(userHandler.Profile)))

    // Apply global middleware
    handler := middleware.Chain(mux,
        middleware.Recovery,
        middleware.Logging(logger),
        middleware.CORS,
    )

    server := &http.Server{
        Addr:         ":" + cfg.Port,
        Handler:      handler,
        ReadTimeout:  cfg.ReadTimeout,
        WriteTimeout: cfg.WriteTimeout,
    }

    // Graceful shutdown
    go func() {
        slog.Info("server starting", "port", cfg.Port)
        if err := server.ListenAndServe(); err != http.ErrServerClosed {
            slog.Error("server error", "err", err)
            os.Exit(1)
        }
    }()

    quit := make(chan os.Signal, 1)
    signal.Notify(quit, syscall.SIGINT, syscall.SIGTERM)
    <-quit

    slog.Info("shutting down...")
    ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
    defer cancel()

    if err := server.Shutdown(ctx); err != nil {
        slog.Error("forced shutdown", "err", err)
    }
    slog.Info("server stopped")
}
```
